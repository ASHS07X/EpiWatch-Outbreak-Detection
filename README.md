EpiWatch: Disease Outbreak Prediction System

EpiWatch is a modern healthcare analytics dashboard that visualizes and analyzes disease outbreak data using CSV datasets.

Live Website: https://epiwatch-dash.lovable.app/

 Features

-  KPI Metrics (Cases, Deaths, Recoveries)
-  Interactive Charts (Line & Bar)
-  Heatmap Visualization (Leaflet.js)
-  Filters (Region & Date)
-  Alerts for High-Risk Areas
-  CSV Upload & Data Processing

Dataset

This project uses the **OWID COVID-19 dataset**:

- File: `owid-covid-data.csv`
- Source: https://ourworldindata.org/covid-deaths

Technologies Used

- HTML, CSS, JavaScript
- Chart.js
- Leaflet.js
- Git LFS (for large dataset handling)
